Ansible

Ansible é software utilizado para gerenciamento de configurações, implantação de softwares e como orquestrador em geral. Existem outros softwares similares (puppet e saltstack por exemplo), porém diferentemente do ansible estas soluções precisam de um aplicativo cliente instalado em cada máquina que será gerenciada. Já o Ansible utiliza ssh para gerenciar computadores linux e winrm para hosts windows. Apesar de não necessitar de um cliente são necessárias algumas configurações no winrm para que o ansible seja capaz de gerenciar adequadamente uma máquina windows. Para realizar as tarefas nos computadores desejados, o ansible utiliza scripts escritos em YAML que são chamados Playbooks.

Exemplo de playbook: 

                
                  ---
                  - hosts: droplets
                  tasks:
                    - name: Installs nginx web server
                      apt: pkg=nginx state=installed update_cache=true
                      notify:
                        - start nginx

                  handlers:
                    - name: start nginx
                    service: name=nginx state=starte

Além disso, ansible é uma ferramenta opensource desenvolvida pela redhat.
